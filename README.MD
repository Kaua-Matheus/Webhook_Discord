# Como Rodar?
- Crie um `.env` seguindo o que está em `.env.example`;
- Instale o python;
- É altamente recomendado criar um ambiente virtual:
    `python3 -m venv .venv`
- Ative o ambiente:
    Linux: `source .venv/bin/activate`
    Windows: `.\venv\Scripts\activate`
- Instale os requisitos:
    `pip3 install -r requirements.txt`

Na pasta principal do projeto, use o seguinte comando para rodar o projeto:
`python3 main.py`


# How to Run?

- Install python;

    Windows: `.\venv\Scripts\activate`
- Install requirements:
    `pip3 install -r requirements.txt`
